# redia-aeditor

> Redia WYSIWYG

## Install

```bash
npm install --save redia-aeditor
```

## Usage

```tsx
import React, { Component } from 'react'

import Aeditor from 'redia-aeditor'
import 'redia-aeditor/dist/index.css'

class Example extends Component {
  render() {
    return <Aeditor />
  }
}
```
